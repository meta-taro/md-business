md-business で指定できる項目

Markdown 先頭の --- 〜 ---（frontmatter）と品目名に 以下が書けます。

登録番号: "T1234567890123" （適格請求書発行事業者）
適格請求書発行事業者の登録番号（T + 13 桁）。 国税庁の公表サイトで発番後に記入します。
免税事業者: true （免税事業者モード）
適格請求書発行事業者の登録を受けていない発行元の場合、登録番号の代わりに このフラグを true にしてください。 タイトル直下に 「※ 適格請求書ではありません」 を、本文末尾に インボイス制度経過措置（2023年10月〜2029年9月）の案内を自動出力します。 手書きで備考に書く必要はありません。
fileName: "御請求書_{請求先}{敬称}_{発行元}_{YMD}"
PDF 保存時のファイル名テンプレート。{請求先} {発行元} {YMD} {YM} {敬称} が展開されます。
テーマ: "青" （または theme:）
アクセントカラーをプリセットから選択。青 / 赤 / 黄 / オレンジ / 紫 / 黒 / グレー のいずれか、または #2a4d7a のような Hex を直接指定できます。
ロゴ: "data:image/png;base64,..." （または logo:）
会社ロゴを発行元欄に表示。PNG / JPEG / GIF / WebP の data: URL、 または https:// の URL を指定できます。SVG・javascript:・ http:// は安全のため拒否されます。
品目名サフィックス |12px
品目名末尾に |12px / |9pt / |0.8em を付けるとその品目だけ縮小表示します。
例: name: "高車点検改修人員配置|12px"
※ 名前に | をそのまま含めたいときは \| でエスケープ。
stamp.shape: "auto" | "round" | "square"
auto（既定）は法人名（株式会社等）が含まれていれば角印、それ以外は丸印を自動判定します。
stamp.text: "サンプル"
印影の文字列を上書き（既定は発行元名）。
振込先: （または paymentInfo:）
銀行 / 支店 / 種別 / 口座番号 / 名義 を任意で指定。空項目は自動的に省略されます。
軽減税率対象: true （品目内）
8% 対象品目に「※」マークを付けます。税率: 8 でも同等扱い。
小計 / 税額 / 合計の自動計算
taxSummary や totals を書かなくても、items[] から自動計算されます。手入力した場合に差分があれば警告バナーで通知します。

詳細はリポジトリの templates/invoice/standard-ja.md を参照してください。

PDF をダウンロード

Chrome の印刷ダイアログが開きます。
「送信先」を「PDF として保存」 に切り替えて「保存」を押してください。

※ Mac の場合は「PDF として保存」または「PDF に書き出す」と表示されます。
※ PDF にページ番号や URL が入る場合は、印刷ダイアログの「詳細設定」を開いて 「ヘッダーとフッター」のチェックを外してください（Chrome が次回以降記憶します）。
