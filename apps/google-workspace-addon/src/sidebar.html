md-business — Markdown ⇔ Sheet

Markdown テーブルとシート間を双方向で変換します。インポートは textarea の内容をシートに書き込み、エクスポートは現在のシートを textarea に Markdown として書き出します。検証シートの場合は セットアップ で列ドロップダウン / 条件付き書式 / 1行目固定を、検証 で値の妥当性を一括チェックします。

GitHub 自動同期（Phase 3C）

frontmatter に repository: owner/repo@branch:path を書いて「有効化」を押すと、シート編集が 2 秒 静まった時点で GitHub に chore(test-spec): sync ... として PUT します。PAT は PdM (担当者) が手作業で登録してください（AI は credential を扱いません）。
